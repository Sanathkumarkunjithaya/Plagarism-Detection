About Us

This is an NLP project created by 6 - AIDS students for assignment purposes under the guidance of Miss. Sandra Jeyes.

Implementation

Our Plagiarism Detector uses Natural Language Processing (NLP) techniques to identify similar text across different documents. It includes features like:

  • Tokenization using NLTK
  • TF-IDF Vectorization
  • Cosine Similarity for comparison
